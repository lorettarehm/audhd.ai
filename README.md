

3. Start the development server:
   ```bash

   ```

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Voice AI**: ElevenLabs Conversational AI
- **Deployment**: Netlify
